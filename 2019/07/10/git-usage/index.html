常用的git命令

Posted on 2019-07-10

  1. git log –pretty=oneline

  2. git log -1

  3. git rev-parse –git-dir #show the .git folder name

  4. git add . git add filename git add all git add . -A

control + r # search the history command

  1. git commit -s

  2. git commit -m “msg”

  3. git commit –amend # add the current file to the previous change list

  4. git push

  5. git bisect 是用二分法协助RD查找是哪笔check-in导致的错误的，请参考

  6. git diff 文件格式说明，link

  7. git diff HEAD – readme.txt

  8. git reset –hard HEAD^ 用remote端最新code盖掉index区域的改动（已经add到index的改动）

  9. git reset –hard 1094a 指定回到某个版本

  10. git cherry-pick

  11. git reflog 查找使用过的git command

Mac01:BlueShell wnnwoo$ git reflog

fcdea80 (origin/master) HEAD@{1}: commit: fix build error

  1. git checkout – readme.txt 丢弃工作区的改动

  2. git reset HEAD readme.txt 暂存区的修改回退到工作区

  3. git diff > abc.diff

  4. git apply abc.diff
